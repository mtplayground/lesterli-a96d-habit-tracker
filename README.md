# Habit Tracker

A Vite, React, TypeScript, and Tailwind CSS starter for a local-first habit
tracker.

## Scripts

- `npm run dev` starts the Vite development server on `0.0.0.0:8080`.
- `npm run build` type-checks and builds the static app.
- `npm run preview` serves the production build on `0.0.0.0:8080`.

## Development

Install dependencies, then start the app:

```bash
npm install
npm run dev
```

The app is currently the baseline shell for issue #1. Habit data models,
storage, routing, and feature components will be added in later issues.
